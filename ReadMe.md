# **Steps to Run**



**1. Install dependencies:**



&nbsp;		*pip install langchain langchain-ollama chromadb gradio pandas*





**2. Install and run Ollama, then pull models:**



&nbsp;		*ollama pull llama3*

		*ollama pull nomic-embed-text*



<b>3.	Update the dataset path in the script.</b>



**4.	Run the application:**



&nbsp;		*python your\_script\_name.py*



5\.	**Open** the Gradio interface OR the generate a local link and start your journey by entering a book preference to receive AI-generated recommendations.
