Steps to Run

1. Install dependencies:

		pip install langchain langchain-ollama chromadb gradio pandas


2. Install and run Ollama, then pull models:

		ollama pull llama3
		ollama pull nomic-embed-text

3.	Update the dataset path in the script.

4.	Run the application:

		python your_script_name.py

5.	Open the Gradio interface OR the generate a local link and start your journey by entering a book preference to receive AI-generated recommendations.
